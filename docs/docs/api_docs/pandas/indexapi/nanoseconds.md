# `pd.TimedeltaIndex.nanoseconds`

`pandasTimedeltaIndex.nanoseconds`

### Example Usage

```py
>>> @bodo.jit
... def f(I):
...   return I.nanoseconds

>>> I = pd.TimedeltaIndex([pd.Timedelta(7, unit="nanos"))])
>>> f(I)
Int64Index([7], dtype='int64')
```
