# `pd.Index.duplicated`


`pandasIndex.duplicated(keep='first')`

```py
  
>>> @bodo.jit
... def f(I):

>>> idx = pd.Index(['a', 'b', None, 'a', 'c', None, 'd', 'b'])

array([False, False, False,  True, False,  True, False,  True])
```
