# `pd.Series.round`

`pandas.Series.round(decimals=0)`

### Supported Arguments

| argument | datatypes                |
|----------|--------------------------|
| `other`  | Series with numeric data |

!!! note
    `Series.round` is only supported on Series of numeric data.
