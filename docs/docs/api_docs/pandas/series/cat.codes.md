# `pd.Series.cat.codes`

`pandas.Series.cat.codes`


!!! note
    If categories cannot be determined at compile time, then Bodo
    defaults to creating codes with an `int64`, which may differ from
    Pandas.


### Example Usage

>>> @bodo.jit
... def f(S):
...     return S.cat.codes
