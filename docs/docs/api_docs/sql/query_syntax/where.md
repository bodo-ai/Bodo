# WHERE



The `#!sql WHERE` clause on columns can be used to filter records that
satisfy specific conditions:

```sql
SELECT <COLUMN_NAMES> FROM <TABLE_NAME> WHERE <CONDITION>
```

For Example:
```sql
SELECT A FROM table1 WHERE B > 4
```
