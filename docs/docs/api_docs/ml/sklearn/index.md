# Scikit-learn

Bodo supports `scikit-learn` versions `1.4.*`. To install scikit-learn in your Bodo environment:

```shell
pip install scikit-learn=='1.4.*'
```
or
```shell
conda install scikit-learn='1.4.*' -c conda-forge
```
