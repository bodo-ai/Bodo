Bodo 2024.2 Release

We are thrilled to announce the Bodo 2024.2 Release, featuring expanded UDF support, Snowflake functions, improved performance, and bug fixes. Key highlights include enhanced compatibility with semi-structured arrays, TIMESTAMP aliases, and implicit lateral joins. Furthermore, we've upgraded Iceberg to version 1.43 for added functionality. In parallel, BodoSQL sees significant improvements in query plans and bug fixes with the release of version 2024.2.1.
